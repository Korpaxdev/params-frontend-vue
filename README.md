# Frontend часть для [API](https://github.com/Korpaxdev/o2rus-taks2-backend-/)

## Варианты запуска:

- Заполняем `.env` файл
- Через Docker - `docker-compose up -d`
- Вручную:
    - Устанавливаем node зависимости `npm install`
    - Запускаем проект `npm run dev`
